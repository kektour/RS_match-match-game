How to play:

Memory is a counter game where the objecr is to find pairs.

When the game begins, all pixtures are hidden.

To Play:

  1. Select two cards to try match the pictures.
  2. If you match the pictures you can go again.
  3. If they don't match it is the computer turn them.
  4. The player that finds all pairs wins!
  5. Have Fun!
